A selection of my open source projects

## gpiozero

gpiozero is a simple interface to GPIO devices with Raspberry Pi. It is in the top 1% downloaded
packages from PyPI.

- [GitHub](https://github.com/gpiozero/gpiozero)
- [PyPI](https://pypi.org/project/gpiozero)
- [ReadTheDocs](https://gpiozero.readthedocs.io/)

## piwheels

piwheels is a Python package repository providing wheels (pre-built binaries) for Raspberry Pi's Arm
architecture, saving users much time and effort.

- [Website](https://www.piwheels.org)
- [GitHub](https://github.com/piwheels/piwheels)
- [ReadTheDocs](https://piwheels.readthedocs.io)

## hostedpi

hostedpi is a Python library and command line interface to the Mythic Beasts Hosted Pi API.

- [GitHub](https://github.com/piwheels/hostedpi)
- [PyPI](https://pypi.org/project/hostedpi)
- [ReadTheDocs](https://hostedpi.readthedocs.io)

## mosromgr

mosromgr (pronounced mos-ro-manager) is a Python library for managing MOS running orders, developed
in my former team at BBC News Labs.

- [GitHub](https://github.com/bbc/mosromgr)
- [PyPI](https://pypi.org/project/mosromgr)
- [ReadTheDocs](https://mosromgr.readthedocs.io)
- [BBC Open Source](https://www.bbc.co.uk/opensource/projects/project/mosromgr)
- [BBC News Labs](https://www.bbc.co.uk/rdnewslabs/projects/mosromgr)

## pyjokes

pyjokes is a Python library and command line program providing one line jokes for programmers. It is

- [GitHub](https://github.com/pyjokes/pyjokes)
- [PyPI](https://pypi.org/project/pyjokes)
